✨ 4 DATABASE FEATURES:

1. PHASE AUTO-DATES (trigger_phase_auto_dates)
   → Auto-timestamps when phase status changes
   → actual_start: triggered on "in_progress"
   → actual_end: triggered on "completed"

2. MILESTONE AUTO-STATUS (trigger_task_milestone_status)
   → Updates milestones when tasks change
   → Recalculates on INSERT, UPDATE, DELETE

3. PROJECT COMPLETE RPC (mark_project_complete)
   → Function wrapper with validation
   → SECURITY DEFINER permissions
   → For finalizing projects

4. DAILY SNAPSHOTS (daily_snapshots + record_daily_snapshot)
   → Records KPI metrics daily
   → Idempotent (safe to re-run)
   → Utility for analytics

OPTION 1: Copy-Paste (Fastest - 5 minutes)
───────────────────────────────────────
1. npm run export:db-polish
2. Select all SQL and copy
3. Go to https://app.supabase.com
4. SQL Editor → New Query
5. Paste SQL → Click RUN

OPTION 2: Automated (Requires Service Role Key)
────────────────────────────────────────────────
1. Get Service Role Key from Supabase Settings
2. Add to .env: SUPABASE_SERVICE_ROLE_KEY=...
3. npm run deploy:supabase:polish
